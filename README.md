# Retail Transaction Dashboard & Customer Analysis

This project is part of my E-Learning journey at MySkill (January 2025), focusing on analyzing retail transaction data and building an interactive dashboard to generate business insights.

## Project Overview
The goal of this project is to explore customer transaction data and transform it into meaningful insights that can support business decision-making. The analysis includes revenue trends, customer behavior, product performance, and payment method analysis.

## Dataset Description
The dataset used in this project consists of retail transaction data, including:

- Customer information (customer_id)
- Transaction details (quantity, revenue, profit, discount)
- Product data (sku_name, category)
- Payment methods (COD, Easypay, Payaxis, etc.)
- Time series data (monthly transactions)

The dataset contains:
- ~3,200+ product records
- ~4,000+ customer records
- Multi-category product transactions

## Dashboard Features
The dashboard provides several key business insights:

### 1. Revenue & Performance Metrics
- Total Revenue: $8.69M
- Total Profit: $2.03M
- Total Customers: ~4K
- Total Quantity Sold: ~13.6K

### 2. Sales Trend Analysis
- Monthly revenue trends (2021–2022)
- Identification of peak sales periods

### 3. Payment Method Analysis
- Distribution of transaction counts
- Contribution of each payment method to revenue

### 4. Product Analysis
- Top product categories by transaction count
- Top categories by transaction value
- Product-level performance

### 5. Discount Analysis
- Discount spending distribution across categories
- Identification of high-discount products

### 6. Customer Insights
- Top customers based on transaction value
- Purchase behavior patterns

## Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization (Looker Studio)
- Excel
- Data Cleaning & Preprocessing

## Key Insights
- Mobile & Tablets dominate transaction value (~46%)
- Payment methods significantly influence transaction distribution
- Certain products generate high profit despite lower transaction counts
- Discount strategies vary significantly across product categories

## Project Outcome
This project demonstrates how raw transaction data can be transformed into actionable business insights through data analysis and visualization.
